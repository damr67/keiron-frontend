
### REQUISITOS
Tener instalado la version del backend localmente. `keiron-backend`

Tener Instalado MySQL Database, puede user XAMPP si desea.


## Antes de Iniciar
### `npm install`


## Iniciar Backend Localmente
### `keiron-backend-$ npm run start`

## Para Iniciar Localmente
### `npm run start`
